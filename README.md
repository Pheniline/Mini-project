---

INTRODUCTION

GridGuard is an IoT-inspired smart-grid monitoring platform designed to simulate real-time transformer monitoring, predictive fault detection, and electricity theft detection.

The system was inspired by real challenges faced by Kenya Power and Lighting Company (KPLC), including:

- Transformer overloads
- Delayed fault detection
- Illegal electricity connections
- Reactive maintenance systems
- Unexpected outages

GridGuard transforms passive transformers into intelligent connected assets capable of:

✅ Monitoring transformer health  
✅ Detecting overload conditions  
✅ Detecting possible electricity theft  
✅ Generating live alerts  
✅ Simulating smart-grid infrastructure  
✅ Providing live monitoring dashboards

---

 Object Oriented Programming Principles Used

~ Encapsulation
Sensitive transformer data is stored securely inside classes.

~ Abstraction
Complex monitoring operations are hidden from the user.

~ Inheritance
Administrative users inherit from base user structures.

~ Polymorphism
Different faults trigger different system responses.

~ Classes & Objects
Transformers, users, alerts, and reports are modeled as objects.
